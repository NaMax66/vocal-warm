# VocalWarm

VocalWarm - простое Nuxt-приложение для вокальной распевки. Оно слушает микрофон, определяет высоту звука и показывает ближайшую ноту.

Основная документация на английском: [README.md](README.md)

## Заметки о проекте

- Папка проекта: `Documents\Codex\VocalWarm`
- Тип приложения: Nuxt 3 single-page app.
- `app.vue` собирает экран и хранит только состояние верхнего уровня.
- Основные UI-блоки лежат в `components/`: `AppHeader.vue`, `StartOverlay.vue`, `PitchReadout.vue`, `TuningMeter.vue`, `PianoKeyboard.vue`, `KeyboardControls.vue` и `VolumeMeter.vue`.
- Общие переводы находятся в `utils/i18n.ts`.
- Настройки CDN для piano samples находятся в `utils/pianoSamples.ts`.
- Общая математика нот и константы диапазона клавиатуры находятся в `composables/useNoteMath.ts`.
- Аудио клавиатуры, Tone.js sampler preload, выбор пресета сэмплов, limiter, громкость инструмента и тайминг отпускания ноты находятся в `composables/useKeyboardAudio.ts`.
- Захват микрофона, autocorrelation pitch detection, состояние распознанной ноты, громкость и статус тюнера находятся в `composables/usePitchDetector.ts`.
- Текущий заголовок на экране: `Распевка по нотам`.
- Язык интерфейса выбирается из языка браузера при первом открытии: русский для `ru-*`, английский для остальных языков.
- Пользователь может вручную переключить интерфейс кнопками `RU` / `EN`; выбор сохраняется в `localStorage` как `vocalwarm-language`.
- Приложение использует Web Audio API и `navigator.mediaDevices.getUserMedia`.
- Воспроизведение нот с клавиатуры использует Tone.js (`tone`).
- Воспроизведение клавиатуры использует Salamander Grand Piano samples через `Tone.Sampler`.
- Сэмплы пианино загружаются из публичных пакетов `@audio-samples/piano-velocity*` на jsDelivr.
- Шестеренка настроек звука под кнопкой `Стоп` позволяет выбрать velocity-слой пианино: `Мягкие` (`velocity1`, +18 dB), `Легкие` (`velocity2`, +14 dB), `Полные` (`velocity8`, +8 dB), `Сильные` (`velocity12`, +4 dB) или `Яркие` (`velocity16`, +2 dB).
- Выбранный пресет сэмплов сохраняется в `localStorage` как `vocalwarm-piano-preset`.
- Пресет по умолчанию - `Яркие` (`velocity16`), если у пользователя нет сохраненного выбора.
- При переключении пресета новый sampler сразу прелоадится, а интерфейс показывает небольшой индикатор загрузки.
- Звук пианино проходит через `Tone.Limiter(-1)`, чтобы уменьшить клиппинг после усиления сэмплов.
- Salamander Grand Piano samples созданы Alexander Holm и лицензированы под CC BY 3.0.
- Клавиатура поддерживает удержание звука, пока удерживается клавиша/указатель, а короткие нажатия звучат минимум около 0.5 секунды перед отпусканием.
- Подсветка нажатой клавиши затухает примерно за 0.5 секунды вместе с коротким удержанием ноты.
- Выбранная нота отмечается на клавиатуре маленьким красным крестиком.
- Блок управления нотой выбирает C2-B6 стрелками; удержание пробела удерживает выбранную ноту.
- Выбранная нота для управления пробелом сохраняется в `localStorage` как `vocalwarm-selected-midi`.
- Горизонтальная прокрутка клавиатуры сохраняется в `localStorage` как `vocalwarm-keyboard-scroll-left`.
- Меню настроек звука закрывается по клику снаружи.
- Pitch detection реализован через autocorrelation по time-domain буферу `AnalyserNode`.
- `AnalyserNode` использует `fftSize = 4096`.
- Для более чистого отслеживания высоты выключены `echoCancellation`, `noiseSuppression` и `autoGainControl`.
- Ноты показываются английскими именами: `C`, `C#`, `D`, `D#`, `E`, `F`, `F#`, `G`, `G#`, `A`, `A#`, `B`.
- Интерфейс показывает ближайшую ноту, октаву, частоту в Hz, отклонение в центах, tuning meter, прокручиваемую клавиатуру C2-B6, volume meter, статус и ошибки микрофона.
- Распознанная нота подсвечивается на клавиатуре, если она находится в диапазоне C2-B6.
- Для доступа к микрофону браузеру нужен `localhost` или HTTPS.
- Последний известный dev URL: `http://127.0.0.1:3000/`.

## Направление разработки

- Держать VocalWarm сфокусированным на вокальной распевке и pitch feedback.
- Предпочитать простой читаемый Vue/Nuxt-код перед добавлением библиотек.
- Полезные будущие функции: упражнения для распевки, более богатые пресеты звучания клавиатуры, история нот, оценка точности, определение диапазона и локализованные названия нот.

## Запуск

```bash
npm install
npm run dev
```

Откройте локальный URL из терминала. Для доступа к микрофону браузеру нужен `localhost` или HTTPS.
